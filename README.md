# Team Task Tracker Application

## Features
- User Management (Create, View, Delete)
- Project Management
- Task Assignment System
- Task Status Tracking (TO_DO, IN_PROGRESS, COMPLETED)

## Tech Stack
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap

## REST APIs
- GET /api/tasks
- POST /api/tasks
- PUT /api/tasks/{id}
- DELETE /api/tasks/{id}

## Tools Used
- Postman (API Testing)
- Git & GitHub
